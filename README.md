## Python Password Manager 

This code can generate and encrypt a random 10 char password (can decrypt too).

Working to upgrade!

### Install guide 

Create a folder

> cd 'nameofyoufolder'

> git clone https://github.com/Mormito/PasswordManager

> pip install cryptography

> python main.py
